About Kojo The Content Mod

Kojo is an AI-powered content moderation system built to protect online communities in real time. It analyses text, images, videos, audio, and URLs through multi-layered safety models which are currently designed to protect online spaces where children are involved (U17). Kojo detects harmful behaviour before it escalates, flags risky patterns, and explains every moderation decision with transparent reasoning.


Kojo works currently works on Discord and the web, with plans to expand. It identifies grooming, harassment, hate, self-harm, violent content, explicit sexual material, scams, malware, and dangerous acts. When needed, Kojo can escalate, time-out users, or automatically remove messages while keeping moderators informed through detailed incident reports.


Unlike keyword-based filters that create false positives or miss things entirely, Kojo is able to focus on intent and context. It tries to understand nuance, evaluate tone, and separate legitimate conversation from harmful behaviour so communities stay safe without punishing normal discussion.

What is Kojo?

Kojo is an AI-driven moderation tool designed to protect communities. Instead of relying on simple keyword filters, Kojo analyses text, links, images, videos, audio, and voice activity to understand intent, context, tone, and risk. It is built to detect grooming, harassment, hate, explicit sexual content, violence, self-harm, scams, malware, and other high-risk behaviour before it escalates.


How Does Kojo Work?

Kojo is invited into your online community like any other bot. Once granted moderation privileges, Kojo monitors text channels, media uploads, links shared by users, andâ€”if your community has public voice channelsâ€”Kojo also monitors voice activity in real time.


Every message, link, or media file is processed through a layered moderation pipeline. Kojo classifies the level of risk, determines the appropriate action, and generates a transparent explanation for moderators. Depending on severity, Kojo may allow the content, warn the user, delete the message, apply timeouts, or escalate immediately for high-risk incidents such as grooming or personal-information solicitation.


All moderation decisions are logged as detailed incident reports for human moderators. These reports include the category, severity, reasoning, confidence score, and any follow-up suggestions. The public web demo uses the same moderation pipeline, allowing anyone to submit messages or media and see exactly how Kojo would respond inside a real Discord server.


Tech Stack

  • Language: JavaScript (Node.js)
  • Discord Bot: Discord.js, Node.js, FFmpeg, OpenAI API
  • Moderation Pipeline: OpenAI (text, image, video, audio, URL analysis), GPT-based classification models, transcription-based voice moderation, frame-level video analysis
  • Backend (Web Demo): Express.js, Node.js, Multer (file uploads), CORS, Dotenv
  • Media Processing: FFmpeg, Fluent-FFmpeg
  • Frontend (Web Demo): HTML, CSS, Vanilla JavaScript
  • Logging & Reporting: Structured JSON incident reports, server-side logging, real-time mod-channel event delivery
  • Infrastructure: Local Node server, environment-based OpenAI key handling (.env)

The Future

Kojo is built to work alongside human moderators, not replace them. As the system grows, the goal is to expand beyond Discord and the web demo into more online environments such as other social platforms, forums, multiplayer gaming, and any digital space where safe interaction matters.


Long term, systems like Kojo could provide real-time safety layers for childrenâ€™s games, public chat systems, and high-traffic online spaces where traditional moderation tools simply cannot keep up. The same multi-modal analysis used can be adapted for platforms with different cultures, rules, and risk profiles.


Kojo is one part of a much larger shift in how online safety will work in the future. We see AI-driven moderation as becoming essential for detecting context, intent and harm at a scale manual teams alone cannot handle. Our focus is to keep improving accuracy, reduce false positives, and build tools that make moderators more effective rather than replacing them entirely.


The future of moderation is hybrid! AI handling the heavy lifting, humans making the final calls. Team Kojo aims to be a reliable part of that future, helping communities stay safe no matter where they exist online.
